FCC Imace Search

This application is querying Google Custom Search API and return results in JSON.
To search head to /search/query followed by your query.
For example /search/beach will return beach images.
It's as well possible to query the last 10 searches here: /latest
This excercise has been developed following FreeCodeCamp challenge: https://www.freecodecamp.com/challenges/image-search-abstraction-layer
